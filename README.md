# Dataset – Percepções sobre o Uso de Redes Sociais

Dados anonimizados utilizados na tese:

"ComuniCanvas: uma ferramenta epistêmica baseada na engenharia semiótica para o design de redes sociais inclusivas"

## Conteúdo

- respostas_formulario.csv

## Descrição

O objetivo foi identificar características do uso de redes sociais por diferentes usuários, especialmente no que se refere aos hábitos de uso, finalidade, barreiras percebidas, controle e personalização na utilização dos recursos disponíveis. Os dados obtidos permitiram identificar alguns padrões e dificuldades relatadas sobre o uso de redes sociais, mas principalmente expuseram problemas que permeiam todos os usuários, independentemente da faixa etária.
